 Math Wizard Hybrid AI

Author: Ali Kahoot
Project Type: Hybrid Math Solver (Deep Learning + Traditional Computation)
Language: Python
Framework: Streamlit, TensorFlow/Keras, NumPy, Matplotlib

> Overview

Math Wizard Hybrid AI is a versatile mathematics application that combines deep learning (DL) and traditional math methods to solve a wide variety of problems. The app is designed to make learning and exploring math fun, interactive, and efficient.

Deep Learning Component: Neural networks are used to predict solutions for linear and quadratic equations.

Traditional Component: Exact calculations for statistics, volume & area, and algebraic expressions are performed using standard Python programming.

This hybrid approach allows users to experience both AI-powered approximation and classical mathematics.

> Features

Linear Equation Solver

Solve ax + b = c using a neural network prediction.

Quadratic Equation Solver

Solve ax² + bx + c = 0 using a deep learning model for approximate solutions.

Statistics Calculator

Calculate Mean, Median, and Mode of a set of numbers.

Plot line graphs for better visualization.

Volume & Area Calculator

Compute volume and surface area for shapes: Cube, Cuboid, Sphere, Cylinder, Cone, Rectangle, and Square.

Algebra Expression Solver

Evaluate algebraic expressions with variables using Python’s evaluation logic.
